# Kerry Vending Machine Design Patterns
### ITTRALEE Year 4 Semester 1 - Software Design Patterns Module
Kerry Vending is a vending machine located in Tralee that dispenses consumer 
goods such as beverage to customers when money is inserted, and the product is
selected from the menu. 

## Design Patterns Used
- Decorator
- Factory Method
- Strategy
- Template Method
- Observer
- Singleton
